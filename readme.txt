== Description ==

Woo Ajax Filter is a plugin allows you create ajax shop. It supports default widget of Woocommerce.

Otherwise we can make ajax function for:

- category

- filter price

- tag

- navigation

- attribues

This plugin is used in C4D Vision - Responsive  WooCommerce Theme

You can view here http://coffee4dev.com/responsive-woocommerce-theme-c4d-vision/

== Installation ==

1. Upload `neo-wc-ajax-filter` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
